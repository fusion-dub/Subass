1. ОПИС ТА МОЖЛИВОСТІ
-------------------------------------------------------------------------------

Subass Notes — це набір інструментів для REAPER, розроблений для автоматизації
дубляжу, озвучення та роботи з субтитрами. Система об'єднує текстовий редактор,
відео-оверлей, менеджер дедлайнів та інструменти лінгвістичної підтримки.

Склад пакету:
- Subass Notes: Центральний хаб для керування проєктом.
- Subass SubOverlay (Lionzz): Візуальне відображення тексту поверх відео.
- Subass Dictionary: База знань, глосарій та власні словники.
- Subass PDF Reader: Читання сценаріїв (pdf, doc) всередині REAPER.
- Imnotbad Notepad: Робочі нотатки.
- Imnotbad Pomodoro: Таймер продуктивності.

Ключові можливості:

- Імпорт SRT та ASS (автоматичний колір регіону для кожного актора).
- Відображення оверлея субтитрів від Lionzz.
- Підтримка наголосів та їх застосування.
- Відкриття ГОРОХу по затисканню на слово.
- Режим режисера (керування правками та зауваженнями).
- Менеджер дедлайнів по проєктам.
- Автопідрізання початку та кінця щойно записаної репліки.
- Відображення асиміляції та евфонічних підказок.
- Перевірка на перегруз для щойно записаної репліки.

Інші можливості:
- PDF рідер.
- Pomodoro та нотатник від imnotbad.
- Можливість автозапуску скрипта при відкритті REAPER.
- Менеджер правок в суфлері.
- Можливість автооновлення.
- Підтримка власних словників.
- Розділення акторів по даберам (додатково відображення конфліктів).
- Відображення наступної репліки з можливістю кастомізації.
- Зворотний таймер та попередження CPS (Characters Per Second).
- Відображення осцилограми на фоні основного текстом.
- Режим "Караоке" (підсвітка слів залежно від гучності/піку аудіо).
- Гнучкі налаштування тексту, кольорів та профілів суфлера.
- Відображення таблиці з усіма репліками: сортування, фільтрація, редагування.
- Розумний парсинг SRT за патернами "(актор): репліка" або "[актор]: репліка".
- Режим "ВЕЛИКИМИ ЛІТЕРАМИ" для зручності читання.
- AI асистент в редакторі тексту.
- Кнопка "#" (в кінці списку таб) для швидкого переходу до вказаної репліки.
- Швидкий вибір акторів в табі Файл.
- Відображення коментарів.
- Збільшення розміру інтерфейсу.
- Режим читача в табі Репліки (таблиці).
- Відображення ім'я актора перед текстом в суфлері.
- Відображення Словника (лайка, сленг ...).
- Розфарбування Item за акторами.
- Режим Слайдера для суфлера.
- Експортування реплік та правок.
- Озвучення виділеного тексту.
- Прибрати дублікати реплік (Waveform Match).
- Нормалізація реплік (EBU R128).
- Глобальний пошук реплік.
- Звуковий глосарій.
- Розрахунок реплік по акторам.
- Відображення % кількості записаних реплік для треку.

2. ІНСТРУКЦІЯ ЗІ ВСТАНОВЛЕННЯ
-------------------------------------------------------------------------------

Запустіть інсталятор для вашої операційної системи який самостійно встановить
всі необхідні файли (macOS_Install.command або Windows_Install.bat).

Інакше встановіть файли вручну:

Для коректної роботи завантажте всі файли в одну папку.

Як запустити в REAPER:
1. Відкрийте "Actions" -> "Show action list".
2. Натисніть "New action" -> "Load ReaScript...".
3. Оберіть файл "plugin/Subass_Notes.lua".
4. Після додавання натисніть "Run" або подвійний клік на назву скрипта.

3. РОБОТА З НАГОЛОСАМИ AI
-------------------------------------------------------------------------------

Використовує нейронну мережу для визначення наголосів за контекстом.
Для роботи потрібно встановити Python 3.9+.

Перший запуск може тривати 2-5 хв (скачування AI-моделей ~500 МБ).

Особливості:
- Наголоси можна застосовувати окремо для вибраних акторів (таба "Файл").
- Підтримка складних слів через AI-асистента.

-------------------------------------------------------------------------------
4. GEMINI AI АСИСТЕНТ
-------------------------------------------------------------------------------

Вбудований асистент використовує нейромережу Google Gemini для допомоги:
- "Перефразувати": створення варіантів репліки.
- "Зробити довше/коротше": адаптація під таймінг.
- "Перевірити наголоси": швидка перевірка конкретного тексту.
- "Тон": зміна настрою (веселіше, серйозніше тощо).

ЯК ОТРИМАТИ КЛЮЧ (БЕЗКОШТОВНО):
1. Перейдіть на сайт Google AI Studio: https://aistudio.google.com/
2. Увійдіть через свій Google-акаунт.
3. Натисніть кнопку "Get API key" у лівому бічному меню.
4. Скопіюйте створений ключ (він безкоштовний для особистого користування).
5. Вставте цей ключ у табі "Settings" плагіна Subass Notes у поле
   "Gemini API Key".

Після цього в текстовому редакторі можна використовувати кнопку "AI".

ВАЖЛИВО !!!
Якщо ви помітили, що асистент часто припускається помилок (галюцинує), можливо,
ваші безкоштовні ліміти Google Gemini вичерпуються. Спробуйте змінити API-ключ
або перейти на платний тарифний план.

5. ДОДАТКОВІ ФУНКЦІЇ ТА "ГАРЯЧІ" КЛАВІШІ
-------------------------------------------------------------------------------

РЕДАКТОР ТЕКСТУ:
- Ctrl+Z: скасування останньої дії (Undo).
- Ctrl+D: зняти виділене тексту.
- Ctrl+Shift+Z: повторення скасованої дії (Redo).
- Ctrl+V / Cmd+V: вставка тексту з буфера.
- Shift+Стрілки: виділення тексту.
- ESC: закрити редактор без збереження.

РЕПЛІКИ (Table):
- Shift+Клік: виділення діапазону реплік у списку.
- Стрілки Вгору/Вниз: навігація за списком реплік.
- Ctrl+Клік: вибір окремих реплік (мультиселекція).
- Ctrl+A: вибрати всі репліки у списку.
- Ctrl+D: знати виділення (якщо декілька), а якщо одна - дублювання репліки.
- Ctrl+Backspace, Backspace / Delete: видалити виділені репліки у списку.
- Фільтрування з повагою до регістра.
- Клік по заголовку: сортування за цим стовпчиком.
- Клік по бургеру "≡": знайти та замінити.
- Клік по бургеру "≡": приховувати колонки в таблиці ("≡" -> Колонки...)
  Червона крапка на кнопці меню сигналізує про приховані колонки.
- Клік по бургеру "≡": здвиг часу для реплік в таблиці.
- Клік по бургеру "≡": режим режисера.

ГЛОБАЛЬНІ ГАРЯЧІ КЛАВІШІ (у будь-якій вкладці):
- Пробіл: Старт/Пауза відтворення REAPER.
- R: Запис.
- Ctrl + S: Зберегти поточні дані.
- Ctrl + Z / Ctrl + Shift + Z: Скасувати / Повернути дію (Undo/Redo).
- 1: Перейти на табу Файл
- 2: Перейти на табу Репліки
- 3: Перейти на табу Суфлер
- 4: Перейти на табу Налаштування
- 5: Відкрити "Мої Дедлайни"
- 6: Розподіл по даберам
- 7: Режим Режисера
- 8: Відобразити SubOverlay
- 9: Пошук слова в ГОРОСі
- 0: Глобальний пошук реплік
- -: Відкрити Нотатник
- =: Відкрити Словник

СУФЛЕР (Prompter):
- Подвійний клік на репліку: швидкий перехід до її редагування.
- Довге натискання на слово (1+ сек): відкриття словника "Горох".
- Асиміляції: плагін автоматично підсвічує фонетичні зміни (наприклад, "ться" 
  відображається як "цця" з хвилястою лінією), якщо ця опція увімкнена в
  налаштуваннях.
- Евфонічні підказки (Милозвучність): підказує правильне чергування
  у/в, і/й, з/із/зі, б/би. Допомагає акторам уникати збігу приголосних.
  Налаштовується в табі "Налаштування".
- Подвійний клік на метадані автоматично копіює час або індек репліки, якщо ця
  опція увімкнена в налаштуваннях.
- Зміна ширини менеджера правок в суфлері.
- Відображення {коментарів} в суфлері при наведенні миші.
  Пунктирна лінія під словом (приклад: "Не смішно{під себе}, народ").
- ПКМ (права кнопка миші): відкрити Оверлей.
- ПКМ (права кнопка миші): відкрити Словник.
- ПКМ (права кнопка миші): відкрити режим Слайдера.

ТАБА ФАЙЛ (File):
- Правий клік на актора:
    * Змінити колір: Вибір індивідуального кольору для візуального розділення.
    * Змінити ім'я: Масове перейменування актора у всіх репліках.
    * Видалити актора: Видаляє актора ТА ВСІ його репліки з проекту.
- Drag & Drop: перетягніть .srt або .ass файл прямо у вікно для імпорту.
- Швидкий вибір: Автоматичний вибір акторів, приймає їх список через кому.
- Застосування наголосів: додавання наголосу для всех реплік вибраних акторів.
- Імпорт правок: через текст (буфер) або файл .csv.
- ПКМ (права кнопка миші) по вільному місцю:
    * WEB-менеджер наголосів: Зовнішній інструмент для роботи з наголосами.
    * Моя Статистика: Відкриває браузерну статистику проекту.
    * Експорт субтитрів: Вибір формату (SRT/ASS).
    * Видалити ВСІ репліки з проекту.

6. РОЗШИРЕНІ МОЖЛИВОСТІ (ADVANCED FEATURES)
-------------------------------------------------------------------------------

--- ДІЇ З ITEM (ACTIONS WITH ITEMS) ---
Цей функціонал допомагає автоматизувати рутинні завдання після запису:
- Нормалізація (Adaptive Limiting): Використовує стандарт EBU R128 для
  вирівнювання гучності з адаптивним обмеженням. Короткі репліки (<2 сек)
  автоматично обмежуються відносно медіани довгих реплік (діалогу), що
  запобігає надмірному підсиленню та зберігає природний баланс гучності.
- Перевірка на кліпінг: Автоматично аналізує записані файли на наявність
  перевантаження. Кліповані айтеми розфарбовуються у ЧЕРВОНИЙ колір, а в
  інтерфейсі з'являється попередження.
- Авто-підрізання (Auto-Trim): При закінченні запису плагін автоматично підрізає
  краї айтема (зазвичай 40-80 мс), щоб прибрати звуки натискання клавіатури або
  кліки миші. Налаштовується у табі "Налаштування".
- Waveform Match: Інтелектуальне порівняння хвиль для виявлення та видалення
  дублікатів реплік.

--- AI ТА ЗОВНІШНІ СЕРВІСИ ---
- Gemini AI Асистент: Доступний у вбудованому текстовому редакторі.
  Допомагає перефразувати репліку, зробити її коротшою або змінити стиль.
  Потребує Gemini API Key.
- ElevenLabs TTS: Витончений синтез мовлення для створення чорнового озвучення
  (scratch track). Дозволяє вибрати голос та миттєво додати його на таймлайн.
- Словник "Горох": Вбудований інструмент для швидкої перевірки значень слів,
  синонімів та наголосів. Викликається довгим натисканням на слово.

7. РЕЖИМ РЕЖИСЕРА (DIRECTOR MODE) — РОБОТА З ПРАВКАМИ
-------------------------------------------------------------------------------

Цей режим призначений для швидкого створення та керування зауваженнями
(правками) безпосередньо під час запису або перегляду матеріалу.

ЯК УВІМКНУТИ:
- Таба "Репліки" -> Кнопка "≡" (меню) -> "Режим режисера".
- З'явиться додаткова панель (знизу або праворуч).

ОСНОВНІ МОЖЛИВОСТІ:
- Швидке створення правок: Введіть текст у поле панелі, і в REAPER миттєво 
  створиться маркер на поточній позиції курсору/відтворення.
- Тегування акторів: Кнопки з іменами акторів дозволяють одним кліком додати 
  тег до правки (наприклад: "[Актор] Текст правки").
- Синхронізація: Панель автоматично відображає текст правки, на якій зараз 
  знаходиться курсор відтворення.
- Керування макетом: Через меню "≡" на панелі можна перемикати розташування 
  (Знизу / Праворуч) або закрити вікно.
- Експорт та Обмін:
    * "Копіювати в буфер": Формує список правок, згрупований за акторами з 
      таймкодами.
    * "Експортувати в CSV": Створення файлу таблиці для подальшої роботи.

ПРИМІТКА ДЛЯ РЕДАКТОРІВ:
Режим режисера підсвічує теги акторів у квадратних дужках зеленим кольором для 
швидкої візуальної орієнтації у списку зауважень.

8. РОЗПОДІЛ ПО ДАБЕРАМ (DUBBER DISTRIBUTION)
-------------------------------------------------------------------------------

Цей інструмент дозволяє організувати та розподілити акторів між даберами 
(виконавцями дубляжу), відстежувати навантаження та експортувати розподіл 
для подальшої роботи.

ЯК ВІДКРИТИ:
- Таба "Файл" -> ПКМ по вільному місцю -> "Розподіл по даберам".
- Відкриється дашборд з двома секціями: "ДАБЕРИ" та "АКТОРИ".

ОСНОВНІ МОЖЛИВОСТІ:

КЕРУВАННЯ ДАБЕРАМИ:
- Додавання дабера: Кнопка "+ Додати дабера" створює нового виконавця.
- Вибір активного дабера: Клік по картці дабера робить його активним для 
  призначення акторів.
- Статистика дабера: Кожна картка показує:
  * Кількість реплік
  * Кількість слів
  * Загальний час озвучки (у форматі М:СС)
  * Кількість призначених акторів
- Контекстне меню (ПКМ по даберу):
  * "Вибрати цього дабера": Автоматично фільтрує список реплік у головному 
    вікні, показуючи лише акторів цього дабера.
  * "Перейменувати": Зміна імені дабера.
  * "Видалити": Видалення дабера та всіх його призначень.

ПРИЗНАЧЕННЯ АКТОРІВ:
- Виберіть активного дабера у верхній секції.
- У секції "АКТОРИ" клікайте по чекбоксах для призначення/зняття акторів.
- Візуальна індикація:
  * Зелений чекбокс: актор призначений активному даберу.
  * Сірий чекбокс: актор не призначений.
  * Блідий вигляд: актор не призначений жодному даберу.
  * Теги у квадратних дужках: показують, яким даберам призначений актор.

СТАТИСТИКА АКТОРА:
Кожна картка актора відображає:
- Кількість реплік
- Кількість слів
- Загальний час озвучки

ЕКСПОРТ РОЗПОДІЛУ:
Кнопка "Копіювати":
- "Короткий (тільки імена)": Простий список у форматі "Дабер: Актор1, Актор2".
- "Розширений (з аналітикою)": Детальна статистика з кількістю реплік, слів 
  та часом для кожного актора.

Кнопка "Експортувати як ASS":
- "Просто експорт": Створює .ass файл з усіма репліками та вбудованими 
  метаданими про розподіл даберів.
- "Експорт з дедлайном": Додає дату дедлайну до назви файлу у форматі 
  [ДД.ММ.РРРР].
- Експортований файл містить:
  * Всі репліки проекту з таймкодами
  * Метадані про призначення акторів (лише для акторів, що існують у проекті)
  * Інформацію лише про даберів з призначеними акторами

ІМПОРТ РОЗПОДІЛУ:
При імпорті .ass файлу з метаданими про даберів:

  лише його акторів.

ВАЖЛИВО:
- Дані про даберів зберігаються ОКРЕМО для кожного проекту REAPER.
- При переключенні між проектами розподіл автоматично завантажується для 
  відповідного проекту.
- Експорт включає лише актуальну інформацію (акторів, що існують у поточному

ТИПОВІ СЦЕНАРІЇ ВИКОРИСТАННЯ:

11. LIONZZ SUB OVERLAY (ОВЕРЛЕЙ СУБТИТРІВ)
-------------------------------------------------------------------------------

Lionzz SubOverlay — це потужний додаток до Subass Notes, який виводить субтитри
поверх відеовікна REAPER у реальному часі, забезпечуючи повний контроль над
контекстом та таймінгом.

ОСНОВНІ МОЖЛИВОСТІ:
- Гнучке відображення: показує поточну репліку, наступну репліку (або дві),
  а також репліки інших акторів (контекст).
- Позиціонування: автоматично "прилипає" до відеовікна REAPER, змінює розмір
  разом з ним (потребує js_ReaScriptAPI). Можливе ручне зміщення.
- Таймінг та Навігація:
  - Прогрес-бар тривалості поточної фрази.
  - Розумний зворотний відлік у паузах (до початку наступної фрази).
  - Кнопка швидкого переходу (>>) до наступної фрази, якщо пауза довга.
- Інтерактивність:
  - Подвійний клік: перехід до редагування тексту в Subass Notes.
  - Затискання на слово: викликає словник Гороха.

ЯК ЗАПУСТИТИ:
- Через контекстне меню Плагіна: Таба "Суфлер" -> ПКМ -> "Відкрити Оверлей".
- Через Action List: знайдіть та запустіть "Subass: SubOverlay (Lionzz)".
- Через Extensions: знайдіть та запустіть "Subass: SubOverlay (Lionzz)".

НАЛАШТУВАННЯ (ПКМ по оверлею):
- Зовнішній вигляд: Шрифти, кольори, тіні, розміри для кожного типу тексту
  (Основний, Наступний, Правки, Інші актори).
- Вирівнювання: Центр/Низ/Верх, відступи.
- Спеціальні опції: "Заповнювати прогалини", "Ігнорувати переноси",
  "Інвертувати Y (macOS fix)".

ПРИМІТКА:
Для коректної роботи прив'язки до відеовікна встановіть розширення
js_ReaScriptAPI через ReaPack (він має автоматично встановитися під час
інсталяції).

12. ЗВУКОВИЙ ГЛОСАРІЙ
-------------------------------------------------------------------------------

Це ваша персональна база звуків та пресетів безпосередньо всередині REAPER.

Для чого це потрібно:
Ця функція незамінна для звукорежисерів при роботі з багатосерійними проєктами.
Якщо у проєктах є повторювані елементи (опенінги, назва серіалу, титри, джингли
тощо), ви можете один раз додати цей уривок в глосарій і потім швидко вставляти
його в усі наступні серії. Вам більше не потрібно відкривати старі проєкти, щоб
знайти та скопіювати потрібний файл.

Як це працює:
- Виберіть потрібний айтем у REAPER.
- Натисніть "Додати з REAPER" у вкладці Глосарію. Скрипт автоматично збереже
  саме обрізану частину айтема.
- Додайте теги та опис для швидкого пошуку.
- Натисніть "Вставити в проєкт", щоб миттєво додати звук на таймлайн у місце
  курсору.

Зручний механізм тегів дозволяє легко фільтрувати базу та миттєво знаходити
потрібний ефект серед сотень доданих записів.

14. ВЕБРОЗРОБКА З НАГОЛОСАМИ (GUI)
-------------------------------------------------------------------------------

Інструмент ukrainian_stress_tool.py має веб-інтерфейс для зручної роботи:
(папка plugin/stress)

ЗАПУСК GUI:
- macOS: Подвійний клік на "Mac_open.command"
- Windows: Подвійний клік на "Windows_open.bat"
- Браузер автоматично відкриється з інтерфейсом

МОЖЛИВОСТІ ВЕБРОЗРОБКИ:
- Текстовий редактор: введіть текст і застосуйте наголоси миттєво
- Обробка файлів: виберіть .srt/.ass/.txt файл, вкажіть назву вихідного файлу
- Автоматичне завантаження: оброблений файл завантажується автоматично

КОМАНДНИЙ РЯДОК:
python3 ukrainian_stress_tool.py "текст"
python3 ukrainian_stress_tool.py input.srt
python3 ukrainian_stress_tool.py input.srt -o output.srt

Для зупинки сервера натисніть Ctrl+C в терміналі.
